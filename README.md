# -Rock-Paper-Scissors-Game
A simple Rock–Paper–Scissors game built using vanilla JavaScript, where a user plays against the computer. The game keeps track of wins, losses, and ties using localStorage, so the score persists even after refreshing the page.
Features

Play using buttons (Rock, Paper, Scissor)

Play using keyboard shortcuts:

R → Rock

P → Paper

S → Scissor

Auto Play mode (computer plays automatically every second)

Persistent score using browser localStorage

Dynamic UI updates without page reload

Clean and beginner-friendly JavaScript logic



Technologies Used

HTML

CSS

JavaScript (Vanilla JS)

Browser LocalStorage
